## Responsive Design

The component adapts to different screen sizes:

### Desktop (>1024px)
- 4-column stat grid
- 2-column detail grids
- Full-width sections

### Tablet (768px - 1024px)
- 2-column stat grid
- 2-column detail grids
- Adjusted padding

### Mobile (<768px)
- Single column layout
- Stacked stats
- Compact spacing
- Touch-friendly buttons
